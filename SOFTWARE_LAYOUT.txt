NAVIGATON - DIE 5 TABS (SIDEBAR)
════════════════════════════════════════════════════════════════════════════════

📊 Dashboard
   → Hauptseite
   → 4 Statistik-Karten
   → 4 Feature-Cards
   → Willkommen-Bereich

📤 Upload
   → Drag & Drop Area
   → Ordner-Button
   → Step-by-Step Anleitung
   → GitHub-Modal

📁 Projekte
   → Projekt-Cards (Grid-Layout)
   → Code-Zeilen, Dateien, Skills
   → GitHub-Info
   → Löschen-Button

📂 Dateien
   → Alle Dateien aufgelistet
   → Mit Icons (HTML 🌐, JS ⚙️, CSS 🎨, etc.)
   → Mit Größe, Typ, Pfad
   → Nach Projekt gruppiert

📈 Analyse
   → Top Skills (mit Progress-Bars)
   → Programmiersprachen
   → Gesamt-Statistiken
   → GitHub Stats
