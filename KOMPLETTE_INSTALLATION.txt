⚡ SCHRITT 1: DATEIEN VORBEREITEN (2 Minuten)
════════════════════════════════════════════════════════════════════════════════

1. Erstelle einen neuen Ordner auf deinem Laptop:

   C:\Users\[DEIN_BENUTZERNAME]\Dokumente\Dev-Portfolio

   (Beispiel: C:\Users\Anjelic\Dokumente\Dev-Portfolio)

   HOW TO: Rechtsklick auf Desktop → Neuer Ordner → Namen eingeben

2. Downloade diese 2 Dateien:
   ✅ dev-portfolio.html
   ✅ script.js

3. Kopiere beide Dateien in den Ordner:
   C:\Users\[DEIN_NAME]\Dokumente\Dev-Portfolio

   ✅ FERTIG! Beide Dateien müssen im SELBEN Ordner sein!

Doppelklick auf: dev-portfolio.html

💥 DIE APP STARTET AUTOMATISCH IM BROWSER!

════════════════════════════════════════════════════════════════════════════════
✨ SO FUNKTIONIERT DIE APP
════════════════════════════════════════════════════════════════════════════════

1️⃣  DASHBOARD-TAB (Automatisch sichtbar)
    - 4 große Statistik-Karten
    - 4 Feature-Beschreibungen
    - Überblick auf einen Blick

2️⃣  UPLOAD-TAB
    - Ziehe einen Projekt-Ordner hierhin
    - ODER klick "Ordner auswählen"
    - System analysiert automatisch
    - GitHub-Modal öffnet sich (optional)

3️⃣  PROJEKTE-TAB
    - Alle deine Projekte mit Details
    - Code-Zeilen, Dateien, Skills, Sprachen
    - GitHub-Info (wenn verknüpft)
    - Löschen-Button

4️⃣  DATEIEN-TAB
    - Alle Dateien schön aufgelistet
    - Mit Icons, Größe, Dateityp
    - Nach Projekt gruppiert

5️⃣  ANALYSE-TAB
    - Top Skills (mit Progress-Bars)
    - Programmiersprachen
    - Gesamtstatistiken


════════════════════════════════════════════════════════════════════════════════
📂 PROJEKT-ORDNER HOCHLADEN - SCHRITT FÜR SCHRITT
════════════════════════════════════════════════════════════════════════════════

1. Klick auf "Upload" Tab (oben links in der Sidebar)

2. Du siehst eine große Upload-Area (mit 📁 Icon)

3. OPTION A - Drag & Drop (einfacher):
   - Öffne File-Explorer
   - Navigiere zu deinem Projekt-Ordner (z.B. C:\...\KasparBot)
   - Ziehe den Ordner auf die Upload-Area in der App
   - Loslassen!

4. OPTION B - Ordner auswählen (Button):
   - Klick "Ordner auswählen" Button
   - Wähle deinen Projekt-Ordner
   - Klick "Öffnen"

5. Wenn Upload fertig:
   ✅ GitHub-Modal öffnet sich
   ✅ Gib GitHub-URL ein (z.B. https://github.com/369Anjelic/KasparHauserBot)
   ✅ ODER klick "Überspringen"

6. Fertig!
   ✅ Gehe zu "Projekte" Tab
   ✅ Dein Projekt wird angezeigt!


════════════════════════════════════════════════════════════════════════════════
🌐 GITHUB VERKNÜPFEN (Optional aber empfohlen)
════════════════════════════════════════════════════════════════════════════════

AUTOMATISCH (nach Upload):
1. GitHub-Modal öffnet sich
2. Gib URL ein: https://github.com/[OWNER]/[REPO]
3. Klick "Verknüpfen"
4. Fertig! Stars, Forks, etc. werden automatisch geholt

MANUELL (später):
1. Gehe zu "Projekte" Tab
2. Suche das Projekt (ohne GitHub)
3. Klick "Repo verknüpfen" Button
4. Gib URL ein
5. Fertig!

OHNE GITHUB:
- App funktioniert perfekt auch ohne GitHub
- Du verlierst nur die GitHub-Statistiken


════════════════════════════════════════════════════════════════════════════════
💡 TIPPS & TRICKS
════════════════════════════════════════════════════════════════════════════════

✅ APP-SHORTCUT ERSTELLEN (für schnellen Zugriff):
   1. Rechtsklick auf dev-portfolio.html
   2. "Verknüpfung erstellen"
   3. Verschiebe die Verknüpfung auf Desktop
   4. Jetzt kannst du App von Desktop starten!

✅ DATEN SIND SICHER:
   - Alle Daten speichern lokal im Browser
   - localStorage = Daten bleiben erhalten
   - Keine Cloud, keine Upload

✅ MEHRERE PROJEKTE:
   - Du kannst unbegrenzt viele Projekte hochladen
   - Jedes wird separat analysiert

✅ PROJEKTE LÖSCHEN:
   - Gehe zu "Projekte" Tab
   - Klick "Löschen" Button
   - Projekt ist weg


════════════════════════════════════════════════════════════════════════════════
🆘 PROBLEME BEHEBEN
════════════════════════════════════════════════════════════════════════════════

❌ APP ÖFFNET SICH NICHT
   → Doppelklick auf dev-portfolio.html
   → Falls immer noch nicht: Rechtsklick → "Öffnen mit" → Browser auswählen
   → Empfohlene Browser: Chrome, Firefox, Edge

❌ SCRIPT.JS WIRD NICHT GEFUNDEN
   → Beide Dateien im SELBEN Ordner?
   → dev-portfolio.html UND script.js müssen zusammen sein!

❌ UPLOAD FUNKTIONIERT NICHT
   → Browser aktualisieren (F5)
   → Andere Browser probieren
   → Ordner-Struktur checken (keine Sonder-Zeichen im Namen)

❌ GITHUB DATEN WERDEN NICHT GEHOLT
   → GitHub-URL muss korrekt sein
   → Beispiel: https://github.com/369Anjelic/KasparHauserBot
   → Nicht: github.com/user/repo (OHNE https://)

❌ DATEN VERSCHWINDEN NACH NEUSTART
   → Browser-Cache löschen nicht vergessen!
   → localStorage speichert die Daten
   → Aber: Browser-Cache löschen = Daten weg!
